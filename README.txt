Crawlr web application
A web app to allow user to create and view pub crawls based in the city of Glasgow.

External sources: 

Google Maps API - https://developers.google.com/maps/
Ryan Devoil's Tango with Django (https://github.com/RyanDevoil/tango_with_django_project)
Google Ajax API (https://ajax.googleapis.com/ajax/libs/jquery/3.0.0/jquery.min.js)
